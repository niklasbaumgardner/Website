About Steganography


Steganography is the practice of hiding secret information within a non secret object. In this example I created, you are able to encode text within an image and then later decode that text from the image. This works by changing each character from the message into it's Unicode value and then converting it to binary. I now have the message in binary and I will iterate through the image change the least significant bit of each RGB value to match the bit of the according bit of the binary message. This changes the message ever so slightly that it isn't noticable by the eye.
