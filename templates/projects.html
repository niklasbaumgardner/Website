Canadian Experience

Canadian Experience is an application that can create a short movie. There are two characters on screen that can be moved independently. The user can drag the characters around the screen and set the frames accordingly. Once done, users can save the file or watch the short movie they created. The application works by interpolationing the position and rotation of each character. There is also two trees in the background start growing at certains frames set by the user. I created this for a school project.

Try It Out

Try It Out is a web application at Humana that developers will upload their API information to so that other teams are able to view and use that API. Try It Out will use the API information to make HTTP requests to the API and will display relevant information to the consumers of Try It Out. Try It Out is used for developers to get introduced to other teams API's and speed up the development process by allowing teams to see the API information by visiting our site instead of having to contact the team and wait for a response. I created this during my internship with Humana.



Who Murdered My Grade?

Who Murdered My Grade? is a Clue inspired remote multiplayer game. Users must create an account and login to play. Games are played with 2-6 users from different browsers. The web application uses websockets to keep each user in the same game state. The goal is to accuse the correct professor of the correct "crime" from the clues. I created this for a group school project.

UML Wars
